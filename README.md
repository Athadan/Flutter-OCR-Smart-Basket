# Flutter-OCR-Smart-Basket
Project Objective: To track prices in real-time during grocery shopping.

Technologies Used:

Flutter & Dart

Google ML Kit (Text Recognition v2)

SQLite (Local Database)

Shutterstock
Discover
FL Chart (Data Visualization)

Features:

Real-time OCR on Camera Stream.

Price/label parsing algorithm using Regex.

Automatic Focus and Zoom control.

Haptic Feedback (Vibration feedback).

Honesty Corner (Challenges & Learnings): 

“This project was developed as a PoC (Proof of Concept). In field tests, it was observed that on-device OCR technology was not stable enough to read product names on cylindrical and shiny packaging in low light conditions. Therefore, the development process was halted.”
